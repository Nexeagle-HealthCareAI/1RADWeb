## Summary

This visual reference shows how the AppointmentBoard adapts across different screen sizes:

- **Mobile:** Single column, stacked components, touch-friendly
- **Tablet:** 2-column grid, responsive wrapping
- **Desktop:** Full table layout, horizontal filters

All layouts maintain the same functionality while optimizing for the device's screen size and interaction method.
