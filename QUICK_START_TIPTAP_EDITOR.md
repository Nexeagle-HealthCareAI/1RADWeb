## 🎨 Toolbar Quick Reference

### Top Row (Left to Right)

1. **↩️ ↪️** - Undo/Redo
2. **B I U S** - Bold, Italic, Underline, Strikethrough
3. **Paragraph ▼** - Headings dropdown (H1-H6)
4. **• 1.** - Bullet list, Numbered list
5. **⬅ ↔ ➡ ⬌** - Text alignment (Left, Center, Right, Justify)
6. **🎨 ✍️** - Text color, Highlight color
7. **▦ 🖼️ ―** - Insert table, Insert image, Horizontal line
8. **🧹** - Clear formatting
9. **💾 Save** - Save button (also Ctrl+S)

## 📝 Common Tasks

### 1. Format Text
```
1. Select text
2. Click B for bold, I for italic, or U for underline
   OR use Ctrl+B, Ctrl+I, Ctrl+U
```

### 2. Create Headings
```
1. Place cursor on line
2. Click "Paragraph" dropdown
3. Select H1, H2, H3, or H4
```

### 3. Create Lists
```
1. Click • for bullet list
2. Click 1. for numbered list
3. Type items, press Enter for new item
4. Press Enter twice to exit list
```

### 4. Change Text Color
```
1. Select text
2. Click 🎨 (color picker)
3. Choose color from dropdown
4. Click "✕ Clear Color" to reset
```

### 5. Add Highlight
```
1. Select text
2. Click ✍️ (highlight picker)
3. Choose highlight color
4. Click "None" to remove highlight
```

### 6. Insert Table
```
1. Click ▦ (table button)
2. Table with 3 rows and 3 columns appears
3. Click in cells to edit
4. Use Tab to move between cells
```

### 7. Insert Image
```
1. Click 🖼️ (image button)
2. Enter image URL in prompt
3. Click OK
4. Image appears in editor
```

### 8. Align Text
```
1. Place cursor in paragraph
2. Click alignment button:
   ⬅ Left
   ↔ Center
   ➡ Right
   ⬌ Justify
```

### 9. Save Report
```
Method 1: Click "💾 Save Draft" button at top
Method 2: Press Ctrl+S
Method 3: Click "💾 Save" button in toolbar
```

## 🎯 Pro Tips

### Tip 1: Use Headings for Structure
```
H2: Main sections (FINDINGS, IMPRESSION)
H3: Subsections (Liver, Spleen, Kidneys)
H4: Sub-subsections (if needed)
```

### Tip 2: Use Colors Wisely
```
🔴 Red: Critical findings
🟠 Orange: Warnings
🔵 Blue: Clinical notes
🟢 Green: Normal findings
🟣 Purple: Special notes
```

### Tip 3: Use Highlights for Review
```
Yellow: Needs review
Green: Confirmed
Blue: Reference
Red: Important
```

### Tip 4: Tables for Measurements
```
Use tables for:
- Lesion measurements
- Organ dimensions
- Multiple findings
- Comparison data
```

### Tip 5: Keyboard Shortcuts Save Time
```
Learn these 3 shortcuts:
1. Ctrl+S (save)
2. Ctrl+B (bold)
3. Ctrl+Z (undo)

These alone will save you hours!
```

## 🔧 Troubleshooting

### Editor Not Showing?
```
1. Refresh the page (F5)
2. Clear browser cache
3. Check browser console for errors
```

### Formatting Not Working?
```
1. Make sure text is selected
2. Try keyboard shortcut instead
3. Check if text is inside a table (some formatting limited in tables)
```

### Save Not Working?
```
1. Check internet connection
2. Try Ctrl+S instead of button
3. Check browser console for errors
```

### Content Disappeared?
```
1. Press Ctrl+Z to undo
2. Check if you're in the right tab
3. Refresh page to reload last saved version
```

## 💡 Best Practices

### For Radiology Reports

1. **Use Headings**
   ```
   H2: CLINICAL HISTORY
   H2: TECHNIQUE
   H2: FINDINGS
   H2: IMPRESSION
   ```

2. **Use Lists for Multiple Findings**
   ```
   FINDINGS:
   • Liver: Normal size and echotexture
   • Gallbladder: No calculi seen
   • Kidneys: Both kidneys normal
   ```

3. **Use Bold for Important Findings**
   ```
   **No evidence of acute pathology**
   ```

4. **Use Colors for Critical Findings**
   ```
   🔴 Suspicious mass in right lobe of liver
   ```

5. **Use Tables for Measurements**
   ```
   | Organ | Size | Notes |
   |-------|------|-------|
   | Liver | 14cm | Normal |
   ```
